Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente cuerpo en formato JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor procesa la petición y responde con un código de estado 200 OK y el siguiente objeto JSON que confirma la recepción de los datos:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Explicación:
- El servidor lee los campos "nombre" y "mensaje" desde el cuerpo de la petición (gracias al middleware express.json() que mapea la información a req.body).
- Luego, devuelve una respuesta estructurada en formato JSON utilizando el método res.json(), confirmando que la información fue recibida correctamente.
